# Boba Bash — hand-drawn UI concept

High-fidelity visual concept for the Boba Bash dashboard in a hand-drawn /
whiteboard style (mockups only — not wired to the app).

- `dashboard.html` — the full dashboard concept (open in a browser, 1600×1000).
  IDE-shaped layout: icon rail · explorer (commit tree, road-to-free-boba) ·
  code editor + "boba radar" world map · terminal · status bar. Uses the real
  product vocabulary: Bashes, RSVPs, site submissions, food balance, live map.
- `art-direction.html` — the design-system sheet: palette (incl. taro / matcha /
  milk tea / tapioca flavor accents), the two brand fonts (Permanent Marker +
  Pangolin, embedded), panel/sticky/icon recipes, and the "cartoony, not messy"
  rules.
- `dashboard-preview.png` — rendered preview of the concept.

Both HTML files are fully self-contained (fonts embedded as data URIs), so they
can be opened from disk, dropped into any static host, or iterated on directly.
Also published to the Claude Design project "Boba Bash — Hand-Drawn Dashboard".
